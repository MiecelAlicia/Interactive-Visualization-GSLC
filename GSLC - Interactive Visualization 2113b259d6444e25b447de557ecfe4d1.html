Tools and Libraries in R for Interactive Visualizations:

  1. Shiny : For building interactive web applications and dashboards.
  1. plotly : For creating interactive plots that can be embedded in web pages.
  1. ggplot2 with ggplotly : To add interactivity to ggplot2 plots.
  1. leaflet : For creating interactive maps.
  1. DT : For creating interactive data tables.

  1. Univariate Graphic (Interactive Histogram)
```{r}
# Creating Interactive Histogram to Visualize the Distribution of Characters' Heights

uni_graph <- plot_ly(starwars, x = ~height, type = 'histogram',                    
	marker = list(color = 'black',                     
	line = list(color = 'grey', width = 2)))

uni_graph <- uni_graph %>%  
	layout(title = 'Distribution of Heights',         
	xaxis = list(title = 'Height (cm)'), yaxis = list(title = 'Count'))
	
uni_graph
```

The output will be:

In R, users can hover over the bars to see exact counts for specific height ranges, providing precise information that a static histogram cannot display directly. Users can zoom in and out to focus on specific parts of the distribution, making it easier to explore and understand the data.

  1. Bivariate Graphic (Interactive Scatter Plot)
# Creating Interactive Scatter Plot to Visualize the Relationship between Height and Mass

bi_graph <- plot_ly(starwars, x = ~height, y = ~mass,
						type = 'scatter',                    
						mode = 'markers', markers = list(size = 10, color = 'blue',                                                    
					  line = list(color = 'black',                                                                 
							width = 1)))

bi_graph <- bi_graph %>% 
						layout(title = 'Height vs Mass',                                
							xaxis = list(title = 'Height (cm)'),                               
							yaxis = list(title = 'Mass (kg)'))
						
bi_graph

The output will be:

Using this interactive scatter plot in R, hovering over points reveals the exact height and mass values along with additional details, which aids in understanding individual data points. Users can also dynamically filter or zoom to explore clusters or outliers, improving the analysis of the relationship between height and mass.

  1. Interactive Maps
```{r}
# Adding fictional coordinates to the dataset
starwars_map <- starwars %>%  filter(!is.na(homeworld)) %>%  mutate(lat = runif(n(), min = -90, max = 90),         lon = runif(n(), min = -180, max = 180))

# Interactive map
leaflet(starwars_map) %>%   
	addTiles() %>%  
	addCircleMarkers(~lon, ~lat, popup = ~paste(name, "<br>", homeworld),                  
				color = 'blue',                   
				fillOpacity = 0.7,                   
				radius = 5) %>%  
	addLegend("bottomright", colors = 'blue', 
				labels = "Characters", 
				title = "Star Wars Characters") 
	
```

The output will be:

By using interactive maps, users can pan and zoom to explore different regions, which is crucial for spatial data analysis. Clicking on markers reveals detailed information about each character and their homeworld, making the map informative and interactive.
